# 气律实验室

## 项目简介

气律实验室是一款用于空气比热容比实验、标准分子模拟和理想气体关系验证的桌面软件。它提供 3D 预览、实时数据、参数调整、实验记录、过程回顾和结果计算等功能。

本仓库是公开发布仓库，只用于提供安装包、自动更新资产和版本说明；不包含应用源代码。

## 下载与安装

请从本仓库的 GitHub Releases 页面下载最新的非预发布版本，除非课程或维护者另有说明。

当前最新稳定版为 [`v5.2.0`](https://github.com/WangJN-ing/hard-sphere-lab-release/releases/tag/v5.2.0)。4.2.3 和 5.1.1 用户可以在应用内的“关于”窗口点击“检查更新”，也可以从 Releases 页面下载 5.2.0 安装包并直接覆盖安装。

Windows 安装包文件名采用 `heat-capacity-lab-setup-x.y.z.exe` 格式。下载后运行安装包即可安装或覆盖安装现有版本。

升级时通常不需要卸载旧版本，也不要清除 `C:\Users\<用户名>\AppData\Roaming\hard-sphere-lab`。5.2.0 保持原 appId、安装身份、更新仓库和用户资料目录，并正式兼容 4.2.3 和 5.1.1 写入的有效工作区。已撤回的 5.1.2 废版实验数据不在兼容承诺内；如果该版本的数据阻止软件启动，请先备份用户资料目录，再重置本地工作区。

## 自动更新

已安装的桌面客户端可以在应用内检查更新。当前版本不会在启动时自动弹出更新提示，请在“帮助”菜单的“关于”窗口中手动点击“检查更新”。自动更新需要同一 GitHub Release 中同时存在 `latest.yml`、安装包 `.exe` 和 `.exe.blockmap`。

如果自动更新失败，可以从本仓库 Releases 页面手动下载最新安装包，并直接覆盖安装现有版本。

## 主要使用流程

1. 新建或打开实验文件。
2. 在左侧栏选择实验文件，并打开需要的实验面板。
3. 在右侧栏调整当前实验参数。
4. 在中心区域运行实验，查看 3D 预览、实时数据和图表。
5. 根据实验类型记录数据、查看关系曲线或完成采样。
6. 实验完成后，从左侧栏打开结果、数据处理或过程回顾。
7. 底部控制台会实时显示运行信息、保存信息、警告和操作反馈。

## 实验模块

### 热容比实验

热容比实验支持演示、引导和自由实验流程。自由实验中可以调整实验参数，并按实验阶段完成打气、升温或恢复、阀门放气和最终恢复等操作。

实验过程中可以记录关键数据。过程回顾会展示实测曲线、理想参考曲线、阶段时间线和记录窗口，便于复盘实验过程。结果页面会计算热容比，并显示误差或得分信息。

### 标准模拟

标准模拟提供硬球分子模拟，用于实时观察分子运动、采样实验数据，并查看对应的结果图表。

### 理想气体模拟

理想气体模拟支持 `P-T`、`P-V` 和 `P-N` 关系验证。用户可以通过关系选择器、扫描变量和采样预设生成数据点，并查看关系验证结果。

## 常见问题

请从本仓库 GitHub Releases 页面下载最新的非预发布安装包。

**自动更新失败怎么办？**
