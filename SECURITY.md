# Security

This repository is a public release repository for installer downloads, auto-update assets, and release notes. It is not an application source repository and is not intended for source review.

For security issues related to released installers or update assets, use GitHub Security Advisories when available. If advisories are not available, open a public issue with only a minimal, non-sensitive description so maintainers can arrange a private reporting channel.

Do not publish exploit details, private data, or sensitive reproduction steps in public issues.
