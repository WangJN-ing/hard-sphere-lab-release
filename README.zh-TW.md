# 熱容比實驗室

## 專案簡介

熱容比實驗室是一款用於空氣比熱容比實驗、標準分子模擬和理想氣體關係驗證的桌面軟體。它提供 3D 預覽、即時資料、參數調整、實驗記錄、過程回顧和結果計算等功能。

本倉庫是公開發布倉庫，只用於提供安裝包、自動更新資產和版本說明；不包含應用程式原始碼。

## 下載與安裝

請從本倉庫的 GitHub Releases 頁面下載最新的非預發布版本，除非課程或維護者另有說明。

Windows 安裝包檔名採用 `heat-capacity-lab-setup-x.y.z.exe` 格式。下載後執行安裝包即可安裝或覆蓋安裝現有版本。

## 自動更新

已安裝的桌面客戶端可以在應用程式內檢查更新。自動更新需要同一 GitHub Release 中同時存在 `latest.yml`、安裝包 `.exe` 和 `.exe.blockmap`。

如果自動更新失敗，可以從本倉庫 Releases 頁面手動下載最新安裝包，並直接覆蓋安裝現有版本。

## 主要使用流程

1. 新建或開啟實驗檔案。
2. 在左側欄選擇實驗檔案，並開啟需要的實驗面板。
3. 在右側欄調整目前實驗參數。
4. 在中心區域執行實驗，查看 3D 預覽、即時資料和圖表。
5. 根據實驗類型記錄資料、查看關係曲線或完成取樣。
6. 實驗完成後，從左側欄開啟結果、資料處理或過程回顧。
7. 底部控制台會即時顯示執行資訊、儲存資訊、警告和操作回饋。

## 實驗模組

### 熱容比實驗

熱容比實驗支援演示、引導和自由實驗流程。自由實驗中可以調整實驗參數，並依實驗階段完成打氣、升溫或恢復、閥門放氣和最終恢復等操作。

實驗過程中可以記錄關鍵資料。過程回顧會展示實測曲線、理想參考曲線、階段時間線和記錄視窗，便於複盤實驗過程。結果頁面會計算熱容比，並顯示誤差或評分資訊。

### 標準模擬

標準模擬提供硬球分子模擬，用於即時觀察分子運動、取樣實驗資料，並查看對應的結果圖表。

### 理想氣體模擬

理想氣體模擬支援 `P-T`、`P-V` 和 `P-N` 關係驗證。使用者可以透過關係選擇器、掃描變數和取樣預設產生資料點，並查看關係驗證結果。

## 常見問題

**在哪裡下載？**  
請從本倉庫 GitHub Releases 頁面下載最新的非預發布安裝包。

**自動更新失敗怎麼辦？**  
請手動下載最新的 `heat-capacity-lab-setup-x.y.z.exe`，然後覆蓋安裝現有版本。

**為什麼 Windows 可能顯示安全警告？**

**如何查看目前應用程式版本？**  
請在應用程式內開啟關於或版本資訊相關入口查看目前版本。

不包含。本倉庫只提供發布資產、自動更新檔案和版本說明。

**在哪裡閱讀版本說明？**  
請查看 `CHANGELOG.md`、`docs/releases/release-notes.json` 或對應 GitHub Release 頁面。
